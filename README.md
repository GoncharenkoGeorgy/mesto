### О проекте

В данном проекте нам необходимо сверстать [макет из Figma](https://www.figma.com/file/StZjf8HnoeLdiXS7dYrLAh/JavaScript.-Sprint-4?node-id=3%3A279&viewport=-271%2C23%2C0.22466860711574554). Для этого нам понадобятся знания

* возможность добавлять и удалять карточки
* настроить просмотр фотографий на весь экран

Для всего этого нам понадобятся знания JavaScript'а
